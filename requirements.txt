#   sqlmap    -> https://sqlmap.org               (apt install sqlmap / git clone)
#   gobuster  -> https://github.com/OJ/gobuster   (apt install gobuster)
#   nikto     -> https://github.com/sullo/nikto   (apt install nikto)
# Scapy additionally requires libpcap for live capture:
#   apt install libpcap-dev  (Debian/Ubuntu)
